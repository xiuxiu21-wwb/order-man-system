# 微信小程序阿里云部署教程

## 一、准备工作

### 1.1 购买阿里云服务器
1. 访问 [阿里云官网](https://www.aliyun.com/)
2. 注册/登录账号
3. 购买 ECS 云服务器
   - 推荐配置：
     - 地域：选择离你用户最近的地区
     - 操作系统：Ubuntu 20.04 或 CentOS 7.9
     - CPU：1 核 2G（测试用）或 2 核 4G（生产用）
     - 带宽：3-5Mbps
4. 购买域名（可选，但建议购买）

### 1.2 配置安全组
在阿里云控制台配置安全组规则：
- 开放端口：
  - 80（HTTP）
  - 443（HTTPS）
  - 22（SSH）
  - 8000（后端 API 端口）
  - 3000（前端端口，可选）

## 二、后端部署（FastAPI）

### 2.1 连接服务器
```bash
ssh root@你的服务器 IP
```

### 2.2 安装必要软件
```bash
# Ubuntu 系统
sudo apt update
sudo apt install -y python3 python3-pip python3-venv git nginx curl

# CentOS 系统
sudo yum install -y python3 python3-pip git nginx curl
```

### 2.3 安装 PM2（进程管理）
```bash
sudo npm install -g pm2
```

### 2.4 上传代码到服务器

#### 方法一：使用 Git
```bash
cd /var/www
sudo git clone 你的代码仓库地址 order-man-system
cd order-man-system
```

#### 方法二：使用 SCP 上传
在本地电脑执行：
```bash
scp -r "c:\Users\20924\Desktop\order man2\ord man" root@你的服务器 IP:/var/www/order-man-system
```

### 2.5 配置 Python 虚拟环境
```bash
cd /var/www/order-man-system
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2.6 配置数据库
```bash
# 初始化数据库
python init_db.py
```

### 2.7 配置环境变量
创建 `.env` 文件：
```bash
nano .env
```

内容：
```env
# 应用配置
APP_NAME=认知障碍老人陪伴与防走失系统
DEBUG=False

# 数据库配置
DATABASE_URL=sqlite:///./example_db.db

# 小程序配置（重要！）
# 这里填写你的阿里云服务器公网 IP
API_BASE_URL=http://你的服务器公网 IP:8000/api

# 高德地图 API
AMAP_API_KEY=82dc66f3753fd5736fe625de1141dafd

# 墨迹天气 API
MOJI_WEATHER_APPCODE=92c76b12080b4d6a88bd90ecf8c73361

# JWT 配置
SECRET_KEY=your-secret-key-change-this-in-production
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
```

### 2.8 使用 PM2 启动后端服务
```bash
pm2 start "uvicorn app.main:app --host 0.0.0.0 --port 8000" --name order-man-backend
pm2 save
pm2 startup
```

### 2.9 配置 Nginx 反向代理
```bash
sudo nano /etc/nginx/sites-available/order-man
```

配置内容：
```nginx
server {
    listen 80;
    server_name 你的域名或服务器 IP;
    
    # 后端 API 代理
    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # WebSocket 支持
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
    
    # 前端静态文件（可选，如果使用 Nginx 托管前端）
    location / {
        root /var/www/order-man-system/frontend/dist;
        try_files $uri $uri/ /index.html;
    }
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/order-man /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

## 四、小程序配置（重要！）

### 4.1 修改小程序 API 地址
在微信开发者工具中，修改 `app.js`：
```javascript
globalData: {
  apiBaseUrl: 'http://你的服务器公网IP:8000/api',
  // ... 其他配置
}
```

### 4.2 配置服务器域名
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入 开发 -> 开发管理 -> 开发设置
3. 服务器域名配置：
   - request 合法域名：`https://你的域名`
   - uploadFile 合法域名：`https://你的域名`
   - downloadFile 合法域名：`https://你的域名`

### 4.3 配置 HTTPS（必须）
小程序要求必须使用 HTTPS，使用 Let's Encrypt 免费证书：

```bash
# 安装 Certbot
sudo apt install certbot python3-certbot-nginx

# 获取证书
sudo certbot --nginx -d 你的域名

# 自动续期
sudo crontab -e
# 添加：0 3 * * * certbot renew --quiet
```

## 六、常见问题

### 6.1 小程序无法连接服务器
- 检查安全组是否开放 8000 端口
- 检查服务器防火墙：`sudo ufw status`
- 确保 API 地址使用 HTTPS

### 6.2 数据库文件权限问题
```bash
sudo chown -R www-data:www-data /var/www/order-man-system
sudo chmod -R 755 /var/www/order-man-system
```

### 6.3 查看日志
```bash
# PM2 日志
pm2 logs order-man-backend

# Nginx 日志
sudo tail -f /var/log/nginx/access.log
sudo tail -f /var/log/nginx/error.log
```

## 七、一键部署脚本

创建 `deploy.sh` 脚本：
```bash
#!/bin/bash

echo "开始部署..."

# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装必要软件
sudo apt install -y python3 python3-pip python3-venv git nginx curl nodejs npm

# 安装 PM2
sudo npm install -g pm2

# 克隆代码
cd /var/www
sudo git clone 你的仓库 order-man-system
cd order-man-system

# 配置 Python 环境
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 初始化数据库
python init_db.py

# 启动服务
pm2 start "uvicorn app.main:app --host 0.0.0.0 --port 8000" --name order-man-backend
pm2 save
pm2 startup

echo "部署完成！"
```

使用：
```bash
chmod +x deploy.sh
./deploy.sh
```
