# 宝塔面板部署指南（图形化界面，无需命令行）

## 🎉 宝塔面板的优势
- ✅ **Web图形界面**，鼠标点击操作
- ✅ 一键安装软件（Nginx、Python、Node.js）
- ✅ 一键部署网站
- ✅ 可视化管理文件
- ✅ 免费SSL证书一键申请
- ✅ **不需要SSH，不需要敲命令**

---

## 🚀 第一步：在阿里云服务器安装宝塔面板

### 1.1 登录阿里云ECS控制台
1. 访问 https://ecs.console.aliyun.com
2. 选择你的云服务器实例

### 1.2 使用云助手安装宝塔
1. 点击实例右侧的 **更多** → **云助手** → **执行命令**
2. 命令类型选择 **Shell脚本**
3. 命令内容粘贴以下代码：

```bash
# Ubuntu/Debian系统安装宝塔
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh ed8484bec
```

如果你的是CentOS系统，用这个：
```bash
# CentOS系统安装宝塔
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec
```

4. 点击 **执行**，等待3-5分钟

### 1.3 获取宝塔登录信息
安装完成后，在云助手的输出结果中会显示：
- **外网面板地址**：http://你的IP:8888/xxxxxx
- **内网面板地址**：http://内网IP:8888/xxxxxx
- **username**：xxxxxx
- **password**：xxxxxx

**保存这些信息！**

---

## 🔧 第二步：配置阿里云安全组

### 开放宝塔面板端口
在阿里云ECS控制台 → 安全组 → 配置规则 → 手动添加：

| 协议端口 | 授权对象 | 说明 |
|---------|---------|------|
| TCP:8888 | 0.0.0.0/0 | 宝塔面板 |
| TCP:888 | 0.0.0.0/0 | phpMyAdmin |
| TCP:21 | 0.0.0.0/0 | FTP |
| TCP:22 | 0.0.0.0/0 | SSH |
| TCP:80 | 0.0.0.0/0 | HTTP |
| TCP:443 | 0.0.0.0/0 | HTTPS |

---

## 🎨 第三步：登录宝塔面板

1. 在浏览器打开宝塔外网面板地址
2. 输入用户名和密码登录
3. 首次登录会推荐安装套件，选择 **LNMP**（Nginx）
4. 选择软件版本：
   - Nginx：1.20+
   - MySQL：5.7（虽然我们用SQLite，但还是装上）
   - PHP：7.4（可选）
5. 点击 **一键安装**，等待10-20分钟

---

## 📦 第四步：安装Python和Node.js

### 4.1 安装Python 3.11
1. 宝塔面板左侧 → **软件商店**
2. 搜索 "python"
3. 找到 **Python项目管理器**，点击 **安装**
4. 安装完成后，点击 **设置** → **版本管理** → 安装 Python 3.11

### 4.2 安装Node.js 18
1. 软件商店 → 搜索 "node"
2. 找到 **Node.js版本管理器**，点击 **安装**
3. 安装完成后，点击 **设置** → 安装 Node.js 18.x

---

## 📁 第五步：上传项目文件

### 方式一：直接上传文件夹（推荐）
1. 宝塔面板左侧 → **文件**
2. 进入 `/www/wwwroot/` 目录
3. 点击 **上传** → 选择你本地的项目文件夹（`ord man`）
4. 等待上传完成

### 方式二：使用Git克隆（需要先上传到GitHub）
1. 软件商店 → 搜索 "git" → 安装 **Git版本控制**
2. 文件 → 进入 `/www/wwwroot/`
3. 点击 **远程下载** → 选择Git
4. 输入你的GitHub仓库地址，点击 **确定**
