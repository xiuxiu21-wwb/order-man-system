# 如何更换 AI 语音陪伴页面背景

## 目录结构
```
miniprogram/
  ├── images/
  │   └── background.jpg    # 背景图片文件
  └── pages/
      └── elder/
          └── voice.wxss    # 样式文件
```

## 更换背景图片步骤

### 方法 1：使用本地图片（推荐）
1. 准备一张背景图片（建议尺寸：750x1334 像素或更大）
2. 将图片命名为 `background.jpg`
3. 放到 `miniprogram/images/` 目录下
4. 在微信开发者工具中重新编译即可

### 方法 2：使用网络图片
如果需要直接使用网络图片，修改 `pages/elder/voice.wxss`：

```css
page {
  height: 100vh;
  background-image: url('https://你的域名.com/background.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
  background-attachment: fixed;
  display: flex;
  flex-direction: column;
}
```

## 图片建议
- **尺寸**：750x1334 像素（适配 iPhone 尺寸）
- **格式**：JPG 或 PNG
- **大小**：建议小于 200KB
- **风格**：温暖、柔和的色调，适合老年人使用
- **内容**：避免过于花哨的图案，保持简洁温馨

## 注意事项
1. 图片文件名必须是 `background.jpg`
2. 图片路径是相对于小程序根目录的绝对路径
3. 修改后需要重新编译小程序才能看到效果
4. 确保图片版权合规

## 当前配置
当前样式已配置为使用本地图片：`/images/background.jpg`

如果图片不存在，页面会显示默认背景色。
