# 阿里云服务器一键部署指南

## 📋 部署前准备

### 服务器选择
- **推荐配置**：2核4G内存，50G云盘，带宽3-5M
- **系统**：Ubuntu 20.04 LTS（稳定可靠）
- **地域**：选择离您用户最近的地域

### 本地准备
- 安装 Git
- 安装 Xshell 或其他 SSH 客户端
- 准备好项目代码（或使用 GitHub 仓库）

## 🚀 部署步骤

### 步骤 1：购买并初始化服务器
1. 登录阿里云控制台
2. 购买 ECS 服务器（选择 Ubuntu 20.04 LTS）
3. 配置安全组，开放以下端口：
   - 80 (HTTP)
   - 443 (HTTPS)
   - 8000 (后端API)
   - 3000 (前端开发)
   - 22 (SSH)
4. 重置服务器密码并获取公网 IP

### 步骤 2：连接服务器
使用 SSH 客户端连接服务器：
```bash
ssh root@您的服务器公网IP
```

### 步骤 3：运行一键部署脚本
在服务器上执行以下命令：

```bash
# 更新系统
apt update && apt upgrade -y

# 安装必要依赖
apt install -y git curl python3-pip npm

# 克隆项目（如果没有代码）
git clone https://github.com/your-repo/elder-care-system.git
cd elder-care-system

# 安装后端依赖
pip3 install -r requirements.txt

# 安装前端依赖
cd frontend
npm install

# 构建前端
npm run build
cd ..

# 启动后端服务
nohup python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload > backend.log 2>&1 &

# 启动前端服务（生产模式）
cd frontend
nohup npm run preview -- --host 0.0.0.0 --port 3000 > frontend.log 2>&1 &
```

### 步骤 4：验证部署
- 后端API：`http://您的服务器公网IP:8000/api`
- 前端页面：`http://您的服务器公网IP:3000`
- 小程序API地址：`http://您的服务器公网IP:8000/api`

### 步骤 5：创建默认用户
```bash
# 在项目根目录执行
python3 -c "from app.db.session import get_db; from app.api.users import create_default_user; db = next(get_db()); create_default_user(db); print('默认用户创建成功')"
```

## 🔧 常用操作

### 查看服务状态
```bash
# 查看后端服务
ps aux | grep uvicorn

# 查看前端服务
ps aux | grep vite
```

### 重启服务
```bash
# 重启后端
pkill -f uvicorn
nohup python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload > backend.log 2>&1 &

# 重启前端
pkill -f vite
cd frontend
nohup npm run preview -- --host 0.0.0.0 --port 3000 > frontend.log 2>&1 &
```

### 查看日志
```bash
# 后端日志
tail -f backend.log

# 前端日志
tail -f frontend/frontend.log
```

## ⚠️ 注意事项

1. **安全组配置**：确保已开放所有必要端口
2. **防火墙**：Ubuntu 默认防火墙已关闭，如启用请配置规则
3. **域名绑定**：如需绑定域名，需要在阿里云控制台配置 A 记录
4. **HTTPS**：生产环境建议配置 SSL 证书
5. **数据库**：项目使用 SQLite，无需额外配置
6. **性能优化**：生产环境建议使用 Gunicorn + Nginx 部署

## 📞 技术支持

如果遇到部署问题，请检查：
- 服务器安全组配置
- 依赖安装是否成功
- 端口是否被占用
- 日志文件中的错误信息
